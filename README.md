# isa-sdk/sdk (PHP)

Unified PHP SDK for [ISA APIs](https://docs.isaapi.com).

This repository is a **read-only mirror**. Do not file PRs here — the canonical source lives at [`Software-Automation-Holdings-LLC/isa-platform`](https://github.com/Software-Automation-Holdings-LLC/isa-platform/tree/main/packages/php), and the `sdk-publish` workflow synchronizes this mirror on every release tag.

## Install

```sh
composer require isa-sdk/sdk
```

See [docs.isaapi.com](https://docs.isaapi.com).
